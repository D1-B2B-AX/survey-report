# 만족도 리포팅 플러그인 — 사용 가이드

## 이게 뭐야?

만족도 설문 raw data를 넣으면, **기업담당자용 + 강사용 메일 초안**을 자동으로 만들어주는 Claude Code 플러그인입니다.

객관식 점수 계산, 주관식 유형 분류, 운영진 의견 초안까지 한 번에 생성하고, HTML 파일로 블록별 검토 + 메일 미리보기를 제공합니다.

---

## 뭐가 필요해?

| 파일 | 필수 여부 | 설명 |
|------|----------|------|
| **만족도 설문 raw data** | 필수 | 엑셀 파일 (.xlsx). 구글 드라이브에만 있으면 먼저 다운로드 |
| **강의관리 시트** | 선택 | 엑셀 파일. 싱크업 문서 내 강의관리 탭도 가능. 없으면 운영진 의견이 일반적인 수준으로 생성됨 |

두 파일 모두 **내 컴퓨터에 저장된 상태**여야 합니다.

---

## 설치 방법

터미널(또는 Claude Code 대화창)에서 아래 명령어를 입력하세요:

```bash
git clone [GitHub URL] ~/.claude/skills/report
```

설치가 완료되면 Claude Code를 열고 `/report` 또는 "만족도 리포팅해줘"라고 입력하면 플러그인이 실행됩니다.
첫 실행 시 필요한 패키지를 자동으로 설치합니다.

---

## 어떻게 써?

### 1단계: 실행

Claude Code에서 아래 중 하나를 입력:
- `/report`
- "만족도 리포팅해줘"
- "만족도 결과보고 작성해줘"
- "설문 결과 정리해줘"

### 2단계: 파일 전달

안내에 따라 **raw data 파일 경로**를 전달합니다.

> 파일 경로 찾는 법: 파일 탐색기에서 파일을 찾고 **Shift + 우클릭** → **'경로로 복사'** → 대화창에 붙여넣기

그 다음 **강의관리 시트**도 전달합니다. 없으면 '없음'이라고 입력.

특이사항이 있으면 함께 알려주세요:
- "강사용만 필요해"
- "1차수만 리포팅할 거야"

### 3단계: 메타 정보 확인

기업명, 과정명, 강사, 교육일시, 응답인원을 자동으로 추출해서 보여줍니다.
맞으면 '확인', 수정할 부분이 있으면 알려주세요.

### 4단계: 자동 생성 (기다리기)

객관식 계산 → 주관식 정리 → 운영진 의견 초안 → 검증까지 자동으로 진행됩니다.
중간에 진행 상황을 알려드립니다.

### 5단계: HTML 확인

다운로드 폴더에 HTML 파일이 생성됩니다. 브라우저에서 열면:
- **블록 검토 탭**: 객관식/주관식/운영진 의견을 영역별로 확인. 근거 자료도 볼 수 있음
- **메일 미리보기 탭**: 실제 메일 형태로 미리보기. "메일 내용 복사" 버튼으로 바로 복사 가능

### 6단계: 수정 또는 완료

- **수정할 부분 있으면**: 대화창에 말하면 됩니다 (예: "운영진 의견 톤 좀 더 부드럽게 해줘")
- **수정 없으면**: 메일 미리보기에서 복사 버튼 누르고, '마침'이라고 입력

---

## 결과물이 뭐야?

HTML 파일 1개가 생성됩니다. 파일명: `기업명_과정명_만족도 리포트.html`

### 블록 검토 탭

| 영역 | 내용 |
|------|------|
| 객관식 | 문항별 평균 점수 + 긍정 비율 (4~5점 기준) |
| 주관식 | 긍정/개선/추가 교육 니즈 유형 분류 + 근거 원문 |
| 운영진 의견 | 기업담당자용/강사용 초안 + 작성 근거 표 |

### 메일 미리보기 탭

| 탭 | 내용 |
|----|------|
| 기업담당자용 | 객관식 표 + 주관식 요약 + 운영진 의견 + 복사 버튼 |
| 강사용 | 동일 구조 (강사 보호 필터링 적용) + 복사 버튼 |

---

## 수정하고 싶으면?

대화창에서 자유롭게 요청하면 됩니다:

| 상황 | 이렇게 말하세요 |
|------|--------------|
| 운영진 의견 톤 변경 | "운영진 의견 좀 더 부드럽게 해줘" |
| 주관식 유형 수정 | "이 유형은 빼줘" / "이 응답은 포함시켜줘" |
| 객관식 표 수정 | "긍정 비율 빼줘" / "순서 바꿔줘" |
| 메일 문구 수정 | "마무리 멘트 바꿔줘" |
| 추가 정보 활용 | "수강자 소속 분포도 넣어줘" |

수정하면 검증 → HTML 재생성까지 자동으로 다시 진행됩니다.

---

## 알아두면 좋은 것

- **객관식 점수는 코드가 계산합니다** — AI가 직접 계산하지 않아서 수치가 정확합니다
- **주관식은 원문 기반입니다** — AI가 없는 내용을 지어내지 않습니다. 블록 검토에서 근거 원문을 확인할 수 있습니다
- **운영진 의견만 AI가 초안을 작성합니다** — 근거 표에서 어떤 자료를 바탕으로 썼는지 확인 가능. 수정이 필요하면 말씀해주세요

## 피드백 / 문의

사용 중 불편한 점이나 개선 의견이 있으시면 아래 방법으로 전달해주세요.

### 피드백 보내는 법

1. Claude Code 대화창에 **'세션 로그 추출'**이라고 입력 → md 파일이 생성됩니다
2. 아래 파일을 **이지현**에게 전달:
   - 추출된 세션 로그 (.md)
   - 사용한 raw data 파일 (.xlsx)
   - 사용한 강의관리 시트 (.xlsx, 있는 경우)
3. 함께 보내주시면 정확한 원인 파악과 개선이 가능합니다

### 담당자

- **이지현** (AX 매니저)
- 플러그인 관련 문의, 개선 요청, 버그 리포트 모두 환영합니다
